# UX 개선 리서치 요약 (2026-02-14)

## 핵심 원칙: Progressive Disclosure (3단계 공개)
1. **Glance (1초)**: 카드에 제목 + 감성 아이콘 + 영향도 바 + "So What?" 한 줄
2. **Scan (10초)**: 카드 확장 → AI 요약 3줄 + 핵심 수치 + 시나리오
3. **Deep Dive (1분+)**: 전체 상세 (타임라인, 시나리오 분석, 원문)

## 금융 대시보드 벤치마크
- **Bloomberg**: 정보 밀도 높되 초기 뷰는 단순, 색상 의미론(빨강=하락, 초록=상승)
- **Koyfin**: 프리셋 뷰 제공 (사용자가 빈 화면에서 뭘 봐야할지 모르는 문제 해결)
- **Simply Wall St Snowflake**: 5축 레이더 차트로 복잡한 데이터 압축
- **Ground News Bias Bar**: 뉴스 출처 분포 시각화 (한 줄 바)
- **Arc Browser**: 호버 프리뷰 (제로 클릭), AI 요약

## "적게 읽고 많이 아는" 패턴
- **Finimize**: "3분이면 충분" + "Why should I care?" 필수 섹션
- **Morning Brew**: 헤드라인이 결론, 본문은 why 설명
- **Tufte Data-Ink Ratio**: 장식 줄이고 데이터 최대화

## 클러스터 맵 시각화
- **카드형 노드** (150x100px+): 원형 노드 대신 정보 담을 수 있는 카드
- **D3-force 물리 시뮬레이션**: 자연스러운 배치, 드래그 가능
- **카드 크기 = 중요도**, 색상 = 감성
- **상위 10-15개만 초기 표시** (34개 모두 보여주면 hairball effect)
- **엣지 두께 = 관계 강도**

## 인포그래픽 패턴
- **Impact Card**: BIG NUMBER + 컬러 바 + 컨텍스트 태그
- **시나리오 카드**: Best/Base/Worst 3칼럼
- **감성 히트맵**: 시간 x 섹터
- **타임라인 플로우**: 뉴스 인과관계를 시간순 시각화

## 상세 페이지 "Intelligence Dossier" 구조
1. **Impact Dashboard (40%)**: 핵심 수치 카드 + 시나리오 분석
2. **AI Narrative (30%)**: 짧고 펀치 있는 3분 요약
3. **Source Flow (30%)**: 뉴스 타임라인 + 원문 링크
